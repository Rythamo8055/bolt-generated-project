Step-by-Step Guide: Setting up Project IDX with Flutter and Cline (Up to Cline Setup)

  1. Access Project IDX and Create a Flutter Project

    • Go to Project IDX: Navigate to the Project IDX website. (Note: you might need to request an invitation and sign in with your Google account)
    • Create New Project: Start a new Flutter project. Select Flutter when prompted.
    • Wait for Project Setup: Project IDX will automatically set up a new development environment.

    It should automatically create a base app, which can be used for creating future prompts and creating new parts.

  2. Explore the Project IDX Interface

    • VS Code-Like Editor: You will be presented with a Visual Studio Code-like interface within your browser.
      • On the left side you will find the explorer, where you will find your files.
      • In the centre is where you find the code editing tab.
    • Built-in Android Emulator: Note that you have a running Android emulator already installed and set up by IDX on the right.

  3. Run the Default Flutter Project

    Project IDX supports directly running the application right of the box with out requiring you to configure anything.

    Run the base application by clicking "play" in the android emulator section.

    This should display a default flutter application.

  4. Install the Cline Extension

    On the left side you will find the extension tab, which is similar to visual studio codes tab.

    • Access Extensions: Click on the Extensions tab.
    • Search for Cline: In the search bar, type in ‚ÄúCline‚Äù.
    • Install Cline: Click the "install" option on the extension.
    • Restart (if needed): If prompted, restart your Project IDX instance.

  5. Access Cline Extension

    Once installed, you‚Äôll find a new Cline icon or pane in the side menu, or a dedicated section on your tab view within Project IDX. You should click this icon.

    This should open up Cline‚Äôs interface on the window that is there for working in project IDX, meaning you won‚Äôt be taken outside the web environment

  6. Configure Cline Model Settings

    • Access Settings: Within the Cline interface, locate the Settings button or menu.
    • Select your Model: Look for a dropdown menu to select a Model.
      • Gemini 1.5 Flash: Choose "Gemini 1.5 Flash" (if available).
      • Other models : (Optional) Other viable options includes "DeepSeek", or Claude.
    • Confirm: Save the configuration.
    • Google AI Studio: You can access Google AI Studio to explore more models.
